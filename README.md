## 🚀 Getting Started

<!-- Prerequisites -->
### 🔧 Prerequisites

- [Nodejs](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)

<!-- Run Locally -->
### 🏃 Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/aadc-bambu.git
```

Go to the project directory

```bash
  cd aadc-bambu
```

Install dependencies

```bash
  yarn
```

Start the local server

```bash
  yarn develop 
```
