<!-- Getting Started -->
## 🚀 Getting Started

<!-- Prerequisites -->
### 🔧 Prerequisites

- [Nodejs](https://nodejs.org/)

<!-- Run Locally -->
### 🏃 Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/aadc-bambu.git
```

Go to the project directory

```bash
  cd aadc-bambu
```

Install dependencies

```bash
  yarn
```

Start the local server

```bash
  yarn develop  
```
